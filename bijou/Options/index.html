Options

Command Line Options

-h
Used to display the help menu
-w
Used to trigger the watch feature which automatically monitors the file system for changes
-i
Used to activate index.html based urls (to browse site without a webserver)
-c
Used to clear/remove the 'site' folder (start fresh as old stuff could hang around in there)
-o:folder
Used specify an output folder (different than site)
-r:folder
Used specify the root path. Used when your files are not located at the website root
-v
Used to enable verbose mode
-d
Used to enable debug mode

Filename

underscore prefixed filenames (_section-file.html)
assembles the file sections in one file
dash in filenames (group-file.html)
will bold (.group CSS) the group part of the file name in the display list
