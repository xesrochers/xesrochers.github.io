Admin Instructions

Changements d'image

  1. Produire 2 images: exemple: image.jpg et image.thumb.jpg
  2. Ajouter l'image dans le répertoire: exemple: img/gallery/seascape
  3. Ajouter la description dans le fichier anglais: exemple: en/fineart/seascapes/fichier.seascape
  4. Ajouter la description dans le fichier français: exemple: fr/beauxart/marin/fichier.seascape
  5. Produire une version locale du site: exemple: make www
  6. Valider les changements à l'aide d'un fureteur: exemple: utiliser chrome/edge/firefox
  7. Enregistrer les changements dans le gestionaire de changements (optionel): exemple: make commit
  8. Deployer le site en production: exemple: make deploy

Changements de page

  1. Naviger vers la page et modifier le fichier: exemple: utilier un éditeur de style Notepad or TextEdit
  2. Produire une version locale du site: exemple: make www
  3. Valider les changements à l'aide d'un fureteur: exemple: utiliser chrome/edge/firefox
  4. Enregistrer les changements dans le gestionaire de changements (optionel): exemple: make commit
  5. Deployer le site en production: exemple: make deploy

IMPORTANT: Porter une attention spéciale au code qui a le format suivant:

  • <wharever> ce sont des code HTML qui ne faut pas changer
  • {$whatever} ce sont des codes pour Bijou
